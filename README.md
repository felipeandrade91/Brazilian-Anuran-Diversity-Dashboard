# Brazilian Anuran Biodiversity Dashboard (Power BI)

Interactive Power BI dashboard exploring the spatial and temporal patterns of Brazilian anuran diversity using GBIF occurrence records.

## Overview

This project analyzes approximately 130,000 occurrence records of Brazilian frogs and toads available through GBIF.

Key features include:

- Taxonomic diversity overview
- Species richness by state
- Temporal trends in biodiversity records
- Seasonal patterns across families
- Geographic distribution analyses
- Species range breadth metrics

## Data source

- Global Biodiversity Information Facility (GBIF)
- Taxonomic scope: Order Anura
- Geographic scope: Brazil
- Extraction date: June 16, 2026
